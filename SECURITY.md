# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Tanso, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email security@tansohq.com with:

- Description of the vulnerability
- Steps to reproduce
- Impact assessment
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a detailed response within 5 business days.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Disclosure Policy

We follow coordinated disclosure. We ask that you give us 90 days to address the issue before public disclosure.
