# Security Policy

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Instead, email **me@dougbaek.com** with details. Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce (or a proof of concept)
- Any affected versions/commits you're aware of

We'll acknowledge your report, investigate, and coordinate a fix and
disclosure timeline with you.

## Supported Versions

Tanso Core does not yet follow a formal versioning/support policy. Security
fixes are applied to the latest commit on `main`.
